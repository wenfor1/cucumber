# CashFlow

CashFlow - це вебзастосунок для обліку особистих фінансів: доходи, витрати, баланс, звіти та профіль користувача.

## Що вміє застосунок

- реєстрація та вхід користувача;
- додавання доходів за категоріями;
- додавання і видалення витрат;
- побудова звітів (день/місяць/рік) з графіком;
- редагування профілю та завантаження аватара;
- збереження банківських карток у профілі;
- темна тема інтерфейсу.

## Технології

- Python
- Django
- SQLite (за замовчуванням)
- Django Templates + Bootstrap 5
- JavaScript (Fetch API)
- Chart.js

## Структура проєкту

- `cashflow/` - налаштування Django-проєкту, глобальні URL, шаблони, static;
- `main/` - бізнес-логіка: models, views, forms, urls, migrations;
- `manage.py` - точка входу для Django-команд;
- `media/` - завантажені користувачами файли (створюється під час роботи);
- `.gitignore` - правила виключення локальних/службових файлів.

## Швидкий запуск (локально)

### 1) Клонувати репозиторій

```bash
git clone https://github.com/wenfor1/cucumber.git
cd cucumber
```

### 2) Створити та активувати віртуальне середовище

Windows (PowerShell):

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

Linux/macOS:

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3) Встановити залежності

Встановіть залежності з файлу `requirements.txt`:

```bash
pip install -r requirements.txt
```

### 4) Застосувати міграції

```bash
python manage.py migrate
```

### 5) Запустити сервер

```bash
python manage.py runserver
```

Після запуску відкрий:

`http://127.0.0.1:8000/`

## Основні сторінки

- `/` - головна;
- `/register/` - реєстрація;
- `/login/` - вхід;
- `/createBudget/` - додавання доходу;
- `/createPlan/` - додавання витрат;
- `/chart/` - звіти і графік;
- `/profile/` - профіль користувача.

## Примітки

- Проєкт орієнтований на локальну/навчальну розробку.
- Перед продакшеном потрібно винести секрети в `.env`, вимкнути `DEBUG`, налаштувати `ALLOWED_HOSTS` і безпечне збереження чутливих даних.
